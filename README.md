# RR Swift Solutions IT Platform

An elegant, highly optimized single-page web platform for **RR Swift Solutions**, specializing in professional **Online Job Support**, functional **IT Training**, staffing, and outsourcing services for software professionals globally.

## 🚀 Key Features
- **IT Services Portfolio**: Structured pages detailing Job Support, Online Training, and IT Consulting.
- **Technology Stacks**: Real-time support details for 25+ technologies (AWS, Salesforce, DevOps, Java Full Stack, Big Data Hadoop, Data Science, Automation Testing, etc.).
- **Dynamic Blog Section**: Articles featuring practical career roadmaps, tools, and support workflows.
- **SEO Optimized**: Dynamic meta tags, OpenGraph descriptors, canonical paths, XML sitemap, and rich JSON-LD configurations for enhanced search ranking.
- **Modern User Experience**: Clean, responsive layout styled with Tailwind CSS and premium glassmorphism accents.

## 🛠️ Tech Stack
- **Framework**: React 18 with Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS & shadcn/ui components
- **Routing**: React Router DOM (v6)
- **SEO Management**: React Helmet Async
- **Unit Testing**: Vitest

## 💻 Getting Started

### Prerequisites
Make sure you have Node.js (v18+) and npm installed.

### Installation
1. Clone this repository locally.
2. Install the dependencies:
   ```bash
   npm install
   ```

### Development
Start the local development server with auto-reloading:
```bash
npm run dev
```

### Build
Generate the optimized production bundle under `/dist`:
```bash
npm run build
```

### Testing
Run the vitest test suite:
```bash
npm run test
```
